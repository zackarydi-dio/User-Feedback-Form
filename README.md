# User Feedback Form

A clean, interactive feedback form built with HTML, CSS, and JavaScript. This project demonstrates form validation, live character counting, tooltip guidance, and client-side display of submitted feedback.

## Features

- Responsive user feedback form layout
- Real-time comment character counter
- Input tooltips for enhanced usability
- Required field validation with user-friendly messaging
- Client-side rendering of submitted feedback entries

## Project Structure

- `index.html` — markup for the feedback form and display area
- `style.css` — styling for form layout, tooltips, validation, and entries
- `cp_1.js` — script for validation, character counting, tooltip behavior, and feedback rendering

## Usage

1. Open `index.html` in a browser.
2. Enter a name, email, and comments.
3. Hover over each field to see helpful guidance.
4. Submit the form to display the feedback below.

## Installation

No installation is required. This is a static web project and runs directly in any modern browser.

## Notes

- The form prevents submission when any field is empty.
- Submitted feedback is displayed immediately on the page.
- The comment field is limited to 200 characters.
